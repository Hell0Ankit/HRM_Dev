# 🚀 HRMDev - Human Resource Management System

HRMDev is a **Django-based Human Resource Management System** designed to manage employees, attendance, leave requests, and holidays efficiently.
It provides **separate dashboards for HR and Employees** with role-based access control.

---

## 📌 Features

### 🔐 Authentication

* HR Registration & Login
* Employee Login
* Role-based access (HR / Employee)

### 🧑‍💼 HR Panel

* Dashboard with analytics
* Manage Employees (Add, Edit, Delete)
* Manage Designations
* View Attendance Reports
* Approve / Reject Leave Requests
* Manage Holiday Calendar

### 👨‍💻 Employee Panel

* Personal Dashboard
* View Profile
* Punch In / Punch Out System
* Apply for Leave
* View Leave Status
* View Holidays

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/Hell0Ankit/HRM_Dev
```

### 2️⃣ Create Virtual Environment

```
python -m venv env
```

### 3️⃣ Activate Environment

**Windows:**

```
env\Scripts\activate
```

**Linux/Mac:**

```
source env/bin/activate
```

### 4️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 5️⃣ Run Migrations

```
python manage.py makemigrations
python manage.py migrate
```

### 6️⃣ Create Superuser

```
python manage.py createsuperuser
```

### 7️⃣ Run Server

```
python manage.py runserver
```
